# Django-Vue

#### 1、开发环境

> 后端：Django 3.1.3
> 前端：Vue 3
> 系统：Win 10
> IDE：Pycharm Pro

#### 2、环境搭建
~~~ sehll
# 创建虚拟环境
python -m venv venv

# 进入虚拟环境
python venv\Scripts\activate.bat

# 安装Django
pip instal django==3.1.3

# 安装 DRF及其依赖库
pip install djangorestframework==3.12.2
pip install markdown==3.3.3
pip install django-filter==2.4.0
~~~
